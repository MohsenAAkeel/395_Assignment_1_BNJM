The HTML Programming Language.

HTML


Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in the web browser. Users can also include Cascading Style Sheets(CSS) and scripting languages(like JavaScript) to supplemnt the web page

HTML was created by Sir Time Berners-Lee in 1991 but it was not released to the public until 1995 as HTML 2.0. There have been many other versions of HTML but the current version is HTML 5.0, which is an extention of HTML 4.0
