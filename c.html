The C Programming Language and Projects that Use it

For the beginner programmer, the C language may be a difficult choice sompared to some othe roptions. For many this is for two resons. Namely, the level of micromanagement and lower level style of the language. For example, memory management is handled entirely by the programmer, and C lacks the "syntactic sugar" that higher levels of abstraction can bring. However, it is these same reasons that make C a great language if you're project requires this.

Maybe you're working on an embedded system that has very low levels of memory. managing that memory may require strict control of its use. C is a language that offers the programmer strict control of the memory his software uses at any point in time.

The ability to micromanage a system is also a benefit for increasing the speed of a program. "Ultra-high frequency" trading software usually makes use of C or C++. C is also a great language to help one learn the inner workings of a modern computer system!
